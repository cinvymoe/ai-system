# 导航到摄像头列表的步骤

## 侧边栏菜单项说明

侧边栏有以下菜单项（从上到下）：

1. **摄像头列表** (`camera-list`) ← 这是你需要点击的！
   - 显示数据库中的所有摄像头
   - 可以添加、编辑、删除摄像头
   - 会调用 `CameraListSettings` 组件

2. **方向摄像头** (`camera`) ← 这个只显示使用说明
   - 显示按运动方向分组的摄像头配置信息
   - 会调用 `CameraSettings` 组件
   - 这个页面不会显示数据库中的摄像头列表

3. 传感器设置
4. AI识别设置
5. 报警日志
6. 管理员管理
7. IPC 通信测试
8. Python 后端测试
9. 关于

## 正确的操作步骤

1. 点击左上角的 **设置图标**（齿轮）
2. 在侧边栏中点击 **第一个菜单项："摄像头列表"**
3. 这时应该会看到：
   - 标题："摄像头列表"
   - "添加摄像头" 按钮
   - 统计信息（总数、在线、离线）
   - 摄像头列表

## 如何确认你在正确的页面

正确的页面应该显示：
- 标题栏：有 "摄像头列表" 文字和 "添加摄像头" 按钮
- 统计卡片：显示总数、在线、离线的数量
- 筛选按钮：全部、在线、离线
- 摄像头列表或"没有找到摄像头"的提示

如果你看到"使用说明"和一些说明文字，那说明你在错误的页面（方向摄像头页面）。
